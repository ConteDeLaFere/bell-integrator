### Задание 4

<p>Напишите функцию, которая проверяет, является ли строка, содержащая только символы (, ), {, }, [, ] сбалансированной.</p>
<p>Сбалансированная строка означает, что:</p>
<ul>
  <li>Каждая открывающая скобка имеет соответствующую закрывающую скобу того же типа.</li>
  <li>Скобки закрываются в правильном порядке.</li>
</ul>
<p>Примеры:</p>
<ul>
  <li>Ввод: "()" → Вывод: true</li>
  <li>Ввод: "({[]})" → Вывод: true</li>
  <li>Ввод: "(]" → Вывод: false</li>
  <li>Ввод: "([)]" → Вывод: false</li>
</ul>


<p>Решение должно использовать <b>стек (Stack)</b> для отслеживания скобок.</p>
